# test-yandex-isaev

## Описание проекта

Это учебный/демонстрационный проект с упором на верстку, адаптивность и оформление страницы. Если нужно, можно расширить его до полноценного многостраничного сайта или добавить интерактивную форму регистрации.

Визуальная концепция включает:

- Hero-секцию с заголовком, описанием и кнопками перехода;
- Блок с информацией о турнире, стоимостью участия и ключевыми мероприятиями;
- Сетку этапов преобразования города;
- Адаптивный дизайн для мобильных и десктопных экранов;
- Использование кастомных шрифтов, SVG/PNG графики и анимаций.

## Структура проекта

- `index.html` — разметка главной страницы;
- `css/style.css` — стили и адаптивная верстка;
- `js/app.js` — интерактивные элементы, включая прокрутку и мобильные карусели;
- `assets/` — изображения, иконки и графические элементы.

## Особенности

- Статический сайт без сборщика и зависимостей;
- Адаптивная верстка под разные разрешения;
- Интерактивные элементы и плавная прокрутка;
- Графические ассеты и оформление, привязанные к тематике шахматного турнира.

## Локальный просмотр

Проект запускается без установки зависимостей. Достаточно открыть `index.html` в браузере или поднять локальный сервер из корня проекта.

Пример через Python:

```bash
python -m http.server 8000
```

После этого открыть в браузере:

- `http://localhost:8000`
